****************************
sunkit-pyvista Documentation
****************************

``sunkit-pyvista`` is a python package for visualizing solar physics data in 3D.
It is built on top of the `pyvista <https://pyvista.org/>`__ package, which itself is built as a Python interface to `VTK <https://vtk.org/>`__.
